## Featured Project

### [Secure Multi-VPC Banking Microservice](https://github.com/ieafraazzz/Secure-Multi-VPC-Banking-Microservice)

A secure, banking-style backend built on **AWS** using a microservices architecture, with strict network isolation via **VPC Peering**, public/private subnet separation, and a disaster-recovery VPC for backups.

- **Transaction Service** — public-facing Flask API, handles validation & routing
- **Account Service** — isolated in a private subnet, manages balances & core logic
- **Backup Service** — isolated in a separate DR VPC, stores immutable transaction logs
- Infrastructure defined with **Terraform**, deployed end-to-end on AWS
